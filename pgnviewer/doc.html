Chess viewer - Sample

Overview

A chess game viewer, displaying the board based on the moves typed on the spreadsheet. This visualization is a thin wrapper around LT-PGN-VIEWER 3.4 chess game viewer (http://www.lutanho.net/pgn/pgnviewer.html).

Example

Sample Code

<html>
  <head>
    <script type="text/javascript" src="http://visapi-gadgets.googlecode.com/svn/trunk/pgnviewer/chessviewer.js"></script>
    <script type="text/javascript" src="http://www.google.com/jsapi"></script>
  </head>
  <body>
    <iframe id="board" style="width: 500px"  scrolling="no"></iframe>
    <script type="text/javascript">
      google.load("visualization", "1");
      google.setOnLoadCallback(drawBoard);
	  var board;
	  function drawBoard() {
	    data = new google.visualization.DataTable();
	    data.addColumn('string', 'White');
	    data.addColumn('string', 'Black');
	    data.addRows(3);
	    data.setValue(0, 0, 'g4');
	    data.setValue(0, 1, 'e5');
	    data.setValue(1, 1, 'b4');
	    data.setValue(1, 2, 'b5!');
	    data.setValue(2, 0, 'f4');
	    data.setValue(2, 1, 'Qh4+');
	  
	    var ifr = document.getElementById('board');
	    var board = new Chess(ifr);
	    board.draw(data, options);
      }
    </script>
  </body>
</html>

Expected Data Format

Two columns with text. The first column is the moves of the white player, and the second is of the black player. There is no limit to the number of rows. The moves format is the common chess format, as described in http://en.wikipedia.org/wiki/Algebraic_chess_notation.
