About the visapi-gadgets Project

The visapi-gadgets project is an open-source project that provides date visualizations that use the Google Visualization API. These visualizations can be used to visualize data in a web page, or in a Google Spreadsheet.

Visualizations

  • Bars of Stuff
  • Bowling
  • Chess Viewer
  • Piles of Money
  • Term Cloud
  • Word Cloud
