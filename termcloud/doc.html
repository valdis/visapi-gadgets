TermCloud - Sample

Overview

A list of terms, where the size and color of each word is determined by a specified frequency value (typically the number of times it appears in some text).

Example

Sample Code

<html>
  <head>
    <link rel="stylesheet" type="text/css" href="http://visapi-gadgets.googlecode.com/svn/trunk/termcloud/tc.css"/>
    <script type="text/javascript" src="http://visapi-gadgets.googlecode.com/svn/trunk/termcloud/tc.js"></script>
    <script type="text/javascript" src="http://www.google.com/jsapi"></script>
  </head>
  <body>
    <div id="tcdiv"></div>
    <script type="text/javascript">
      google.load("visualization", "1");
      google.setOnLoadCallback(draw);
      function draw() {
        data = new google.visualization.DataTable();
        data.addColumn('string', 'Label');
        data.addColumn('number', 'Value');
        data.addColumn('string', 'Link');
        data.addRows(3);
        data.setValue(0, 0, 'First Term');
        data.setValue(0, 1, 10);
        data.setValue(1, 0, 'Second');
        data.setValue(1, 1, 30);
        data.setValue(1, 2, 'http://www.google.com');
        data.setValue(2, 0, 'Third');
        data.setValue(2, 1, 20);
        var outputDiv = document.getElementById('tcdiv');
        var tc = new TermCloud(outputDiv);
        tc.draw(data, null);
      }
    </script>
  </body>
</html>

Expected Data Format

The first column is a string with the term. The second column is a positive number with the freqency of this term, typically the number of times the term appears in some text. A third optional column is a string with a link (url) that should be opened when the term is clicked.

Options

target Target for the links. default is '_top' which means the main current page. Use '_blank' to open links in a new tab.

Methods

getSelection - Standard selection. Only one row element can be selected.

Events

select - Standard selection event
